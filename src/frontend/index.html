About Pocket Forecaster

Purpose: Pocket Forecaster is an intelligent smartphone recommendation system that suggests phones based on your budget, usage type, and OS preference.

Sentiment Analysis: Reviews are analyzed in real-time using a hybrid NLP model combining VADER and BERT to classify sentiments as Positive, Neutral, or Negative.

Key Highlights:

  • Java backend with Spark API
  • MVC & Strategy design patterns
  • Responsive frontend using HTML, CSS, JS
  • Dynamic recommendations with accordion panels
  • Bonus suggestions and real-time sentiment visualization
